Removal
1. Remove instrument panel. Dashboard / Instrument Panel
2. Remove housing support bracket.
3. Disconnect heater core hoses and plug heater core outlets (to prevent coolant spills).
4. Remove heater box housing.

Installation
1. Install seals around perimeter of heater core so there are no gaps, and glue in place.
2. If heater core does not lock into place during installation, fasten to heater box using screws through tabs on top of heater core.





3. When replacing the heater core (1) with fresh/recirculating air flap: Install vacuum hose adaptor (2) on new heater core.
4. Replace gasket (3).
5. Remainder of installation is the reverse of removal.
