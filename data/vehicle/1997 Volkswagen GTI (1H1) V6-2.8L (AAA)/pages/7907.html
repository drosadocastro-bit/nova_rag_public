Power Mirror Switch: Service and Repair

Adjustment Switch -E43- with Mirror Selector Switch -E48-:





Removing:
- Carefully pry off door handle trim.
- Remove mirror switch connector.
- Disengage switch retaining tabs (arrows) and remove switch from trim.

Installing:
- Install switch into door handle trim until retaining tabs engage.
- Install switch connector.
- Install door handle trim.
