Description of On-Board Diagnostics II

Function
The Motronic Engine Control Module (ECM) -J220- for the fuel injection and ignition system is equipped with a Diagnostic Trouble Code (DTC) memory.

If malfunctions occur in the sensors and components being monitored, they will be stored in the DTC memory together with an indication of the type of malfunction.

The stored malfunctions are displayed after initiating the malfunction display. Check DTC Memory (scan tool function 02)

After malfunctions have been eliminated, DTC memory must be erased, and the repair must be verified via the appropriate display group. With Manufacturer's Scan Tool (VAG)

The readiness code must be created again after every time the DTC memory is erased and when the voltage supply to ECM -J220- is interrupted. Monitors, Trips, Drive Cycles and Readiness Codes
