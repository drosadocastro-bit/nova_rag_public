Checking Rear Window Wiper/Washer System

Use tester VAG 1466 when troubleshooting rear window wiper/washer system!

Test Conditions:

- Fuses No. 4 and 50K!
- Rear window wiper in end position!

Troubleshooting preparations:

- Remove rear window wiper/washer system relay (J 30), stamped with production control number 72, from relay position 2
- Remove wiper/washer intermittent relay (J 31) stamped with production control number 19 or 99 (depending on equipment), from relay position 8.
- Insert test adaptor VAG 1466/3 into relay position 2.
- Connect VAG 1466 to test adaptor.
- Switch ignition ON.

Test procedure information:
- Perform test steps according to table.
- If specified value/test result is obtained, continue troubleshooting with next test step (test result OK).
- If specified value/test result is not obtained, locate and eliminate malfunction using table.
- Use valid wiring diagram for checking wiring.
- Remove all test wires from sockets of VAG 1466 after each test step.

IF MALFUNCTION HAS NOT BEEN ELIMINATED AFTER PERFORMING TEST STEPS 1 THROUGH 5, REPLACE REAR WINDOW WIPER/WASHER RELAY (J30)!

After repairing malfunction, check rear window wiper/washer system for proper operation.
