Warranty - Transmission Manufacture Date, Code Letters

Number: V 97-05

Date: 3/1/1997

Subject:
Transmission Date of Manufacture and Code Letters

In an effort to expand our reporting to Germany, it is now necessary to indicate the Code letters and Date of Manufacture in the comments section of your warranty claim when the transmission is replaced.





The Code Letters and Date of Manufacture can be found on the transmission housing; the Code letters of the transmission also appear on the vehicle data plates.
