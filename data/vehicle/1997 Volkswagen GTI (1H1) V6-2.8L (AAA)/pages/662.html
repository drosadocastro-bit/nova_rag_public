General Information

NOTE:
- Hose connections are secured with either screw-type, spring-type or locking hose clamps.
- Always replace locking clamps with spring-type or screw-type clamps.
- Fuel hoses in engine compartment must only be secured with spring-type clamps. The use to locking clamps or screw-type clamps is not permitted.
- Install the spring-type clamps with commercial spring-clamp pliers, Hazet 798-5 or equivalent.
- The control module for the fuel injection and ignition system is equipped with a Diagnostic Trouble Code (DTC) memory Before carrying out repairs, adjustment work and trouble shooting DTC memory must be checked and vacuum connections checked (outside air).
- For trouble-free operation of the electrical components, a voltage of at least 11.5 volts is necessary.
- Do not use contact sprays or similar agents in the area of the connector for the HO2S. To function correctly HO2S requires reference air, which they receive via the connector. If contact spray gets into the HO2S via this path it will lead to damage/ malfunctions.
- During some checks it can happen that the Engine Control Module (ECM) will recognize and store a malfunction. Therefore after completing all checks and repairs the DTC memory must be checked and if necessary erased.
