Function 03-Output Diagnostic Test Mode



Output Diagnostic Test Mode (DTM) (VAG 1551 Scan Tool Function 03)

The output diagnostic test mode is part of the ABS electrical test. The ABS pump motor and the functioning of the hydraulic circuits (brake line connections to wheel brake calipers; valve functions) can be checked for leaks or for incorrect connections using the output diagnostic test mode.

NOTE:
^ The vehicle must be raised until all wheels are free to turn. (second technician required to rotate wheels).
^ The -C- button can be used to exit the test sequence at any time.
^ After depressing the brake pedal several times the vacuum in the brake servo will be exhausted, therefore more pressure must be applied to the brake pedal in order to attain the same fluid pressure in brake system as that attained with vacuum.
^ When the vacuum for the brake servo is exhausted it can lead to a situation in which the wheels do not lock; start the engine and build-up vacuum in the brake servo.

EXAMPLE (left-front wheel, FL):

- Indicated on display.

Output diagnostic test mode
IFL:VBAT OFL:OV Wheel FL locked

WHERE;
- IFL = IInlet valve Front Left
- VBAT = Voltage BATtery; voltage at valve.
- 0FL = 0utlet valve Front Left
- 0V = 0 Volts; no voltage at valve.
- Locked/free = Wheel condition.

Caution! Wheel condition (locked or free) must be checked and verified by a second technician.

- Indicated on display:

Output diagnostic test mode
EDL valves/Hyd-P:VBAT Wheel FL/FR locked

WHERE;
- Hydr-P = Hydraulic Pump
