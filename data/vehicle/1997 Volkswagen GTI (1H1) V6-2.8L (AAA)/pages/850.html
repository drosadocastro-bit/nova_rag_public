096 Transmission

DIAGNOSTIC TROUBLE CODE MEMORY, ERASE

Requirements

- Diagnostic Trouble Code (DTC) Memory has been checked and codes displayed.
- After last DTC has been displayed, Press --> key.

After DTC Memory has been checked

^ Indicated on display:

Rapid data transfer HELP
Select function XX

- Press buttons 0 and 5.

(This selects function 05, "Erase DTC Memory").

^ Indicated on display:

Rapid data transfer Q
05 Erase DTC memory

- Press Q button to confirm entry.

^ Indicated on display:

Attention!
DTC memory was not interrogated

NOTE: If the ignition was switched OFF or the engine was left RUNNING, between checking and erasing DTC Memory, then the DTC Memory will not be erased.

Adhere strictly to the sequence of operations, i.e. always check DTC Memory first before attempting to erase.

^ Indicated on display:

Rapid data transfer -->
DTC memory is erased

(DTC Memory will be erased approx. 5 seconds after the message is displayed.)

- DTC Memory is now erased.

NOTE: Wait about 1 minute before checking the DTC Memory again.

^ Indicated on display:

System cannot be interrogated!

^ Printout with printer switched ON:

1 DTC Recognized
00811 3333
System cannot be interrogated

- Transmission Control Module (TCM) -J217- given too little time to recognize malfunctions.

Wait about 1 minute before checking the DTC Memory again.

After checking and erasing the DTC Memory, carry out a test drive and check the DTC Memory again.

- When the DTC Memory is interrogated, the following message should be displayed:

"No DTC recognized"
