Identifying Part Number on Coil Spring

If the color code is not recognizable, then the spring part number can be determined from a code stamped on the upper coil spring.

- Example: VW/Audi logo plus "191 4 S"

^ 191 = Type
^ 4 = Middle group of part number (411)
^ S = Index according to the nine digit part number
^ Complete part number is 191 411 105 S


- Example: VW/Audi logo plus "191 45 AB"

^ 191 = Type
^ 4 = Middle group of part number (411)
^ 5 = Last number of the number group 105
^ S = Index according to the nine digit part number
^ Complete part number is 191 411 105 AB
