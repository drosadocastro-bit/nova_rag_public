Diagnostic Trouble Code (DTC) General Information

WARNING:
- Check the airbag system wiring by visual inspection only
- Do not check continuity or resistance of the airbag ignitor circuits.
- Switch off ignition before checking routing of wiring.

The airbag control module -J234- is located behind the center console at the end of the tunnel. It is equipped with Diagnostic Trouble Code (DTC) memory. The Data Link Connector (DLC) is located behind a cover in the instrument panel, just below the heating and ventilation controls.

The airbag control module -J224- detects malfunctions in the airbag system and stores them in permanent memory.

Also, malfunctions are stored that may be caused by intermittent interruptions in the wiring or by poor connections. These malfunctions are called 'sporadic" and are indicated by the letters "SP";

When the ignition is switched on, the airbag Malfunction Indicator Lamp (MIL) -K75- comes on for approximately 4 sec. and then goes out again. If the airbag MIL continues to blink for additional 15 seconds it indicates that the airbag for the passenger is locked (will not be deployed).

- If the airbag MIL -K75- does not go out after approx. 4 sec., the voltage supply to the airbag control module -J234- has been interrupted. Check DTC memory and correct malfunctions.
- If the airbag MIL -K75- stays on, the voltage supply of the airbag control module -J224- has been interrupted. Check DTC memory and correct malfunctions.
- If the airbag MIL -K75- flashes continuously, the airbag control module -J224- is faulty and must be replaced.

To begin troubleshooting the airbag system, initiate the On Board Diagnostic (OBD) program and check for stored Diagnostic Trouble Code (DTC) information using the VAG 1551 Scan Tool.

The displayed malfunctions information is used to refer to the Diagnostic Trouble Code (DTC) table for notes on possible causes and directions for specific repair procedures.
