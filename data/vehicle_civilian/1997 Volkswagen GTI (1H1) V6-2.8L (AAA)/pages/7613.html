CD Changer - Installation Instruction

Group: 91

Number: 96-05

Date: Dec. 15, 1996

File this Technical Bulletin in Repair Manual No: W42 011 294 117

Subject:
CD Changer, Installation Instructions

Model(s):
Golf, GTI, Cabrio, Jetta 1995 -->

Supersedes TB. 91-96--03, dated Sept. 15, 1996 (Bulletin cover page only! Contents remain)

Note:
Golf CD changer installation information also applies to Cabrio models.

These pages (not included) contain factory approved service installation instructions for Volkswagen-Clarion CD changers into models stated above equipped with Premium Radios Part No. 1HM 035 186 C or D.

This information is new and will be integrated into the next Repair Manual update.

Notes:

Only use the following factory developed CD changer kits. Separate Golf and Jetta kits contain a Clarion CD changer, changer cable and model specific mounting brackets. Kit Part Numbers:

Golf, GTI, Cabrio.CD changer/installation kit

^ Part No. 1HM 098 111

Jetta CD changer/installation kit

^ Part No. 1HM 098 111A





This information supersedes the installation instructions included with the CD changer kit.

Caution!

Part numbers are for reference only, Always check with your Parts Dept. for the latest parts information.
