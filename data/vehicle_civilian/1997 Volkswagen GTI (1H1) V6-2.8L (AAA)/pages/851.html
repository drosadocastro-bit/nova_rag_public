01M Transmission

REQUIREMENT

^ Diagnostic Trouble Code (DTC) Memory interrogated. How to Display and Read Diagnostic Trouble Codes

DTC MEMORY INTERROGATED

^ Indicated on display:

Rapid data transfer HELP
Select function XX

- Press keys 0 and 5.
- The function "Erase DTC Memory" is entered with 05.

^ Indicated on display:

Rapid data transfer Q
05 - Erase DTC Memory

- Confirm entry with key Q.

^ Indicated on display:

Attention!
DTC Memory not interrogated

NOTE:
- If the ignition was switched off, e.g. between interrogating the DTC Memory and erasing DTC Memory, the DTC Memory will not be erased.

- Adhere strictly to the sequence of operations, i.e. first of all interrogate DTC Memory.

^ Indicated on display:

Rapid data transfer -->
DTC Memory is erased

- The DTC Memory is now erased.

NOTE:
- The DTC Memory will be erased approx. 5 seconds after the display appears.

- Wait about 1 minute before interrogating the DTC Memory again.

^ Indicated on display:

System cannot be interrogated!

^ Print out with printer switched on:

1 DTC recognized !
00811 3333
System cannot be interrogated

- Transmission Control Module (TCM) -J217- given no time to recognize DTCs.
- Wait about 1 minute before again interrogating the DTC Memory.
- After interrogating and erasing the DTC Memory, test drive the vehicle and again interrogate the DTC Memory.
- When the DTC Memory is interrogated, the following display should appear:

"No DTC recognized!"
