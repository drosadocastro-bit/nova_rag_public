096 Transmission

SCAN TOOL V.A.G 1551 - CONNECTING AND SELECTING FUNCTIONS

Requirements

- Battery Positive Voltage (B+) OK.
- Fuses 14 (S14), 16 (S16) and 21 (S21) are OK.
- Selector lever in position "P" and parking brake applied.

Check the following Ground (GND) connections for corrosion and poor contact; repair if necessary:

- Transmission GND connections.
- GND attaching point, located at left of relay panel.
- Battery GND strap.
- GND strap between battery and transmission.

Connecting VAG 1551 Scan Tool

DLC Location:





Remove ashtray and slide cover for 16 Pin Data Link Connector (DLC) to the left (see arrow).

With ignition switched OFF, connect VAG 1551 Scan Tool (ST) with VAG 1551/3 adapter cable.

^ Indicated on display:

VAG-Self Diagnostics HELP
1-Rapid data transfer

^ Operating modes -1- and -2- are displayed alternately.

VAG-Self Diagnostics HELP
2-Flash code output

- Additional operating instructions can be printed out by pressing the HELP button on the VAG 1551 Scan Tool.
- The --> button is used to advance the Scan Tool program sequence.
- An automatic check can be carried out in the "Rapid data transfer" operating mode. Then all vehicle control units will be interrogated automatically.

VAG 1551 Scan Tool Operating Instructions

Switch ON ignition.

Switch ON printer with the PRINT button (indicator lamp in button lights up).

Press button 1 for "Rapid data transfer" mode.

^ Indicated on display:

Rapid data transfer HELP
02-Enter address word XX

Press buttons 0 and 2 (this selects program 02 for "transmission electronics").

^ Indicated on display:

Rapid data transfer Q
02-Transmission electronics

Press Q button to confirm input.

^ Indicated on display:

096927731AD AG4 Gearbox 096 1283
Coding 00000 WSC 131071

The Transmission Control Module (TCM) identification, TCM coding and VAG 1551 Scan Tool dealership number are displayed.

Transmission Control Module (TCM) Identification

Depending upon version, the Transmission Control Module (TCM) can indicate a different identification number from that shown in the example.

- 096927731 AD: Part number
- AG4 Gearbox 096: 4-speed automatic transmission 096
- 1283: EPROM (Program level)
- Coding 00000: Not required for current versions
- WSC 131071: Dealership number (example) of the VAG 1551 Scan Tool (ST) with which the most recent coding was performed.

If display appears as shown below:

^ Indicated on display:

Control module does not respond! HELP

- Press the HELP button, a list of possible malfunction causes will be printed out.
- After eliminating the possible causes of malfunctions, again enter the address word 02 for "Transmission electronics" and press Q to confirm.

If "Control module does not answer!" again appears:

- Check wiring connections for Data Link Connectors (DLC's).
- Check Diagnostic Trouble Code (DTC) 65535, "control module faulty!"
