Electronic Component Replacement

NOTE: Before removing battery ground, access trouble codes, note all radio station pre-sets and obtain radio security code.

All Engine Control Module (ECM) memory will be erased if ground cable is removed (including all adaptive functions).

1. Before servicing electronic components, disconnect the battery ground unless specific replacement procedures advise otherwise.

2. Fuel and Engine Control System wiring should be at least 4 inches away from ignition wires.
