Function 07-Code Control Module



Code Control Module (VAG 1551 Scan Tool Function 07)

The ABS control module in the vehicle is coded for the specific vehicle application. New ABS control modules from the parts system are not coded and must be coded after installation. Coding is only possible after the dealership number has been entered into VAG 1551 scan tool.

TEST SEQUENCE
1. Connect VAG 1551 scan tool and select "Brake Electronics" address word 03 with ignition switched on.

NOTE: For scan tool connection instructions refer to Connecting and Selecting Function under Scan Tool Operation.

- Indicated on display:

Rapid data transfer HELP
Select function XX

2. Press buttons -0- and -7- to select "Code Control Module" function 07, and press -Q- button to confirm input.

- Indicated on display:

Rapid data transfer HELP
07-Code Control Module

3. Press -Q- button to confirm input.

- Indicated on display:

Code control module
Enter code number XXXXX

4. Enter relevant code number for vehicle and press -Q- button to confirm input.

Code number Vehicle type
03604 Golf, GTI, Jetta, Cabrio
04505 Passat

NOTE: Always refer to the Technical Service Bulletins for the lastest information on control module codes.

- Indicated on display:

Coding 03604 WSC XXXXX

NOTE: WSC XXXXX, where XXXXX indicates the dealer code. Refer to documentation for the VAG 1551 or equivalent scan tool.

5. Press right ARROW button.

- Indicated on display:

Rapid data transfer HELP
Select function XX

6. Select function 06 End output. Function 06-End Output

NOTE:
^ If the ABS control module has been incorrectly coded, the ABS warning light -K47- and the warning light for brake system -K118- will remain on constantly.
^ If the ABS control module is not coded (Code XXXXX) or has been incorrectly coded, the ABS warning light -K47-and the warning light for brake system -K118- will flash (once per second).
