Fluid - Differential

TYPE SLF [1]


REMARKS:
[1]Do not use Dexron-III. In applications where SLF is recommended, verify fluid installed before topping off.

The 01M transmission, final drive and planetary gearbox are filled with VW Automatic Transmission Fluid (ATF) which has a yellowish color.

The ATF dipstick has been discontinued. The ATF level is checked using a new procedure.

Checking the ATF level or changing the ATF is now performed during the inspection service.

CAUTION: Only VW ATF must be used. Only the VW ATF available as a replacement part can be used in the automatic gearbox 01M.

Container size: 0.5 Liter
Part No.: G 052 162 A1

Container size: 1.0 Liter
Part No.: G 052 162 A2
