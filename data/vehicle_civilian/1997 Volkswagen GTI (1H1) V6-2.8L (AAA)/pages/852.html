096 Transmission

DIAGNOSTIC TROUBLE CODE MEMORY, DISPLAY

- Connect Scan Tool VAG 1551 and enter address code "02", Transmission electronics, and advance until "Select function XX" appears as in the display.

^ Indicated on display:

Rapid data transfer HELP
Select function XX

- Press buttons 0 and 2 again.

(The function "Check DTC Memory" is selected with 02).

^ Indicated on display:

Rapid data transfer Q
02-Check DTC memory

- Press Q button to confirm entry.

- The number of stored malfunctions or "No malfunction recognized" appears in the display.

- The stored malfunctions are displayed one after another and printed out.

^ Indicated on display:

2 DTC's recognized!
(Example Only)

- Press --> button.

- After the last malfunction has been displayed and printed out, the malfunctions should be corrected as described in the Diagnostic Trouble Code (DTC) charts.

- Press --> button.

^ Indicated on display:

Rapid data transfer HELP
Select function XX

NOTE: After checking the DTC Memory and correcting the malfunctions. erase the DTC Memory.
