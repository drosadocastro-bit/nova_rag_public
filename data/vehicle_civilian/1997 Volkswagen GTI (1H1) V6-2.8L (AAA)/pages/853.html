01M Transmission

DIAGNOSTIC TROUBLE CODE (DTC) MEMORY - DISPLAY

- Connect Scan Tool (ST) VAG 1551 and enter address word "02 Transmission electronics" and advance until "Select function XX" appears in the display.

^ Indicated on display:

Rapid data transfer HELP
Select function XX

- Press keys 0 and 2.
- The function "Check DTC Memory" is selected with 02.

^ Indicated on display:

Rapid data transfer Q
02 - Check DTC Memory

- Confirm entry with key Q.
- The number of stored malfunctions or "No DTC recognized" appears in the display.
- The stored malfunctions are displayed in turn and printed out.
- Press --> key.
- After the last malfunction has been displayed and printed out, the malfunctions should be corrected as described in the DTC charts. Testing and Inspection
- Press --> key.

^ Indicated on display:

Rapid data transfer HELP
Select function XX

NOTE: After checking the DTC Memory and correcting the malfunctions, erase the DTC Memory. How to Clear Diagnostic Trouble Codes
